# JaimeMUSIK

A lightweight, custom audio player widget designed to overlay seamlessly onto the Schoology platform interface.

## Installation


*Note: You need a userscript manager like Tampermonkey or Robomonkey installed in your browser first.*

## Features
- **Integrated UI:** Compact control overlay panel positioned inside the Schoology sidebar.
- **Playback Controls:** Fully functional pause, play, and dual-directional 10-second time skips.
- **Dynamic Tracking:** Interactive visual timeline progress bar synchronized with track length.
- **Lyrics Display:** Text container layer mapped to real-time audio playback steps.
